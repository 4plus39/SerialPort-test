# modem_test
modem_test is a tool for communication from SUT serial port to modem.

## Environment
language:	Python 2.7.15rc1

os: Ubunru 18.04

## Installation
Use the package manager pip to install pyserial.

    pip install pyserial

## execution
### Text mode
    sudo python main.py

### GUI mode
    sudo python mainUI.py
